# 以图搜图 Skill

当用户上传图片并希望查找来源时：

1. 获取用户上传的图片路径
2. 调用 `reverse_image_search(path)` 工具
3. 向用户展示搜索结果，包括相关链接和描述
4. 若未找到结果，使用 `web_search` 补充搜索图片文件名相关信息

**注意：** 该功能基于文件名和关键词搜索，实际以图搜图的准确度取决于搜索引擎的支持情况。
