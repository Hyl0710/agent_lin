# 以图搜图 Skill

当用户上传图片并希望查找来源时：

1. 获取用户上传的图片路径
2. 调用 `reverse_image_search(path)` 工具
   - 内部通过 SauceNAO API 做真正的视觉相似匹配
   - SauceNAO 不可用时自动降级为文件名搜索
3. 向用户展示搜索结果，包括相似度分数、来源链接和描述
4. 若未找到结果，使用 `web_search` 补充搜索图片文件名相关信息

**注意：** 该功能通过 SauceNAO 实现真正的以图搜图，非简单的文件名搜索。
每日有调用次数限制（匿名约 20 次/天），超出后自动降级。
